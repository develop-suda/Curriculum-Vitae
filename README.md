### 🏠 自宅ネットワークインフラ構築プロジェクト（個人学習）

#### 期間 : 2ヶ月間（段階的構築中）

#### プロジェクト概要
ネットワーク技術への理解を深めるため、自宅環境にて複数のネットワークサーバを統合構築。

#### 構築環境・使用技術
- **ハードウェア**: 中古ノートPC（コスト効率重視）
- **OS**: Ubuntu Server
- **構築サービス**:
  - **Squidプロキシサーバ**: URLフィルタリング・ACL設定による通信制御
  - **OpenVPNサーバ**: 証明書認証方式によるセキュアVPN接続環境
  - **BIND DNSサーバ**: ドメイン名前解決サービス（段階的構築中）

#### 技術的成果・学習内容
- **ネットワーク概念の具現化**: 抽象的だったネットワーク構造を実際の設定・構築を通じて体系的に理解
- **トラブルシューティング能力**: 
  - VPN接続時のNAT設定問題を独力で原因特定・解決
  - 内部ネットワーク接続は成功するが外部接続ができない問題の解析・対処
- **統合サーバ管理**: 単一環境での複数サービス運用によるリソース管理・最適化スキル習得
- **自主学習能力**: 純粋な技術的興味から始まった継続的学習姿勢の実証
