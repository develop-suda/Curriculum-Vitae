#### 担当業務

- **PCキッティング作業**：約300台規模の効率的な作業プロセス設計・実行
- **プロジェクト管理**：更新スケジュール調整・日程管理・進捗追跡
- **インフラ業務**：物理的なPC交換作業・環境移行サポート
- **トラブルシューティング**：ネットワーク設定問題の原因特定・解決

#### 技術的成果・学び

- **大規模プロジェクト管理**：物量の多いプロジェクトでの作業効率化手法とリスク管理を習得
- **インフラ技術理解**：社内ネットワーク環境の理解とハードウェア管理スキルを実践的に習得  
- **問題解決能力**：固定IP設定時のLAN接続問題など予期しないトラブルへの対応力を強化
- **コミュニケーション能力**：多部署との調整・利用者サポートを通じた業務推進力を向上

#### 業務に関連した個人学習での技術習得

プロジェクトでネットワークに対する理解の重要性を感じ、自宅環境でネットワークサーバ構築を実践。
**2ヶ月間の段階的構築**により、以下の技術を習得：

**🔧 構築環境・技術**
- **ハードウェア**: 中古ノートPC + Ubuntu Server
- **Squidプロキシサーバ**: URLフィルタリング・ACL設定による通信制御
- **OpenVPNサーバ**: 証明書認証方式によるセキュアVPN接続環境
- **BIND DNSサーバ**: ドメイン名前解決サービス（段階的構築中）

**📈 学習成果**
- **ネットワーク概念の具現化**: 抽象的だった概念を実際の設定・構築で体系的に理解
- **トラブルシューティング経験**: VPN接続時のNAT設定問題を独力で原因特定・解決
  - 内部ネットワークには接続できるが外部ネットワークに接続できない問題を解析
  - NATの設定不備が原因であることを特定し、適切な設定により解決
- **統合サーバ管理**: 単一環境での複数サービス運用によるリソース管理スキル習得

**💡 業務への活用**
この学習経験により、業務でのネットワーク関連問題に対してより深い理解と解決能力を発揮。
**アプリケーション開発とインフラ技術を組み合わせたフルスタックエンジニア**としての成長を実現し、
今後のプロジェクトでもネットワーク層の知識を活かした包括的な問題解決を目指している。
